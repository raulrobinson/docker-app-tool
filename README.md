# Docker APP Tool

## Description

This tool is a simple application that allows you to manage your docker containers and images. It is a command line tool that allows you to perform the following operations:

- List all containers
- List all images
- Start a container
- Stop a container
- Remove a container
- Remove an image
- Create a container
